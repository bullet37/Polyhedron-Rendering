# Polyhedron Rendering

## Environment

- jdk 1.8 + JavaFx

## Introduction

- This project implements a geometric rendering, using Javafx to support GUI output
- Three-dimensional rotation is possible using the mouse
- Use the scroll wheel to zoom the graph
